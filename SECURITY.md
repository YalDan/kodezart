# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Use one of:

- GitHub's [private vulnerability reporting](https://github.com/YalDan/kodezart/security/advisories/new) (preferred).
- Email the maintainer via the address listed on [github.com/YalDan](https://github.com/YalDan).

Include reproduction steps, affected versions, and impact. You'll get an acknowledgement within 7 days.

## Supported Versions

The project is pre-1.0. Only `main` is supported; security fixes land there.

## Scope

In-scope: code in this repository.
Out-of-scope: third-party dependencies (report upstream), infrastructure not owned by this project.
